# Arduino Piano (WOKWI)

Simulation of a simple piano built with Arduino

## Components

- Arduino Uno
- Pushbutton (12)
- Buzzer

## Features

- One octave piano (C to B)
- Each push button represents a piano key
- Plays the corresponding frequency on the buzzer
- Simple melodies can be played

## Controls

- Press a push button on the breadboard
- The buzzer plays the corresponding note
